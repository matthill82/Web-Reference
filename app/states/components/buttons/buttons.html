Create a button with a button or a element to retain the native click function. Use a disabled attribute when a button can’t be clicked.

Preview

Code

 <button class="stw-button">Button</button>

The base .stw-button looks like a plain text link. It removes all the styling of the native button. It’s typically used to trigger a modal or display a “like” link. All button variations are built by adding another class to .stw-button.

Add the .stw-button--neutral class to create a neutral button, which has a white background and gray border.

Use a neutral icon button is for buttons with an icon on the left or right (not for stateful buttons). Add the .stw-button--neutral class to .stw-button.

The SVG inside receives the .stw-button__icon class. You can position the icon on the right or the left using .stw-button__icon--right or .stw-button__icon--left , which apply the correct amount of space between the icon and the text.

To create the brand button, add the .stw-button--brand class to the .stw-button class.

To create the destructive button, add the .stw-button--destructive class to the .stw-button class.

Use the inverse button on dark backgrounds. Add the .stw-button--inverse class to the .stw-button class.

The stateful button requires the .stw-button--neutral class in addition to the .stw-button class.

The stateful inverse button works just like the stateful button. It requires the .stw-button--inverse class in addition to the .stw-button class.

It uses the class .stw-not-selected in its initial state. When the user activates the button, use JavaScript to toggle the class to .stw-is-selected. The button contains three spans with classes that hide or show the content of the spans based on the class on the button. Each span contains text and a corresponding icon. The SVG will have the .stw-button__icon--stateful class as well as the .stw-button__icon--left class setting the icon on the left.

Stateful icons can be toggled on and off and retain their state. Like stateful buttons, the initial state is .stw-not-selected, and JavaScript is used to toggle it to .stw-is-selected when activated.

Accessibility

For accessibility, include the attribute aria-live="assertive" on the button. The aria-live="assertive" attribute means the value of the &lt;span&gt; inside the button will be spoken whenever it changes.

The various button icons vary in the amount of space they use on the page. Some have background colors, and some are transparent.

  • .stw-button--icon-bare does not take up the 36px height that a normal button does. Although you create it with a ‹button› element, it looks like a plain icon and is the height of the icon.
  • .stw-button--icon-container looks like an icon and takes the default space of a button (36px). It has no background or border.
  • .stw-button--icon-border includes a border. It is the default size (36px).
  • .stw-button--icon-border-filled adds a background color and border like the .stw-button--neutral.
  • .stw-button--icon-small changes the default size of 36px to the size of 32px.
  • .stw-button--icon-x-small changes the default size of 36px to the size of 20px.
  • Use .stw-button__icon--x-small in an.stw-icon--more icon button to indicate that more information is available..
  • Use .stw-button__icon--small within .stw-button__icon--hint on rows in which hovering over the parent element or table row causes the button to brighten up.
  • The normal sized .stw-button__icon is used in many places, including small close icons on alerts.
  • Use .stw-button__icon--large to create a large close button for modals and toasts. It’s usually used with .stw-button--inverse.

The classes added to each button are added to the base .stw-button class. The <svg> inside the buttons all have the base class of .stw-button__icon.

A hint button is grayed out until its parent is hovered over. Then the hover and focus states are just like a regular icon button.

The parent element must have the class .stw-hint-parent, and the icon inside the button has the class .stw-button__icon--hint.

Use the small button with the down icon in rows and action menus. Add the class .stw-button--icon-x-small on the button. Use .stw-button__icon--small for the inside icon.

Accessibility

Icons without identifying text must have descriptive text in a .stw-assistive-text.

Component Overview

Accessibility

If an icon button doesn’t include a label, use a span with .stw-assistive-text to describe the icon for screen readers.
