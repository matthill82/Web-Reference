Code Demonstration

Note: This should only be used within the Web Reference

This web reference includes a directive to show prettified code using Google Code Prettify, so that we can display the code used to create our components.

The prettifier is a directive which can prettify plain-formatted code, this includes plain HTML and Angular expressions.

Showing Syntax

The simplest use of the syntax highlighter is to wrap the code to highlight in a pre tag with the class stw-prettify. The highlighter auto-detects most common languages.

Syntax 
                    

                        console.log("Hello World!");
Output 
                    console.log("Hello World!");
Line Numbering and Language

You can display line numbers for larger blocks of code using the line-nums attribute, and set the syntax highlighting language explicitly using the language attribute.

Languages are specified as the common filename extension. You can see the supported languages here.
