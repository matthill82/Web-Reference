Component Overview

To float an element on the left, add the .stw-float--left class to the element. To float it on the right, add the .stw-float--right class.

Note: The floated element must be first in the document flow, even if it renders on the right.

If you want the parent of the floated element to contain the floats, add the .stw-clearfix class to the parent.
